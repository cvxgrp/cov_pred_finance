covariance.combination.from_ewmas(returns, pairs, min_periods_vola=20, min_periods_cov=20, clip_at=None, mean=False)[source]

Estimate a series of covariance matrices using the iterated EWMA method

param returns: Frame of returns param pairs: list of pairs of EWMA half lives, e.g. [(20, 20), (10, 50), (20, 60)],

pair[0] is the half life for volatility estimation pair[1] is the half life for covariance estimation

param min_periods_vola: minimum number of observations to start EWMA for volatility estimation (optional) param min_periods_cov: minimum number of observations to start EWMA for covariance estimation (optional) param clip_at: clip volatility adjusted returns at +- clip_at (optional) param mean: subtract EWMA mean from returns and volatility adjusted returns (optional)

return: Yields tuples with time, mean, covariance matrix, weights
